## 📋 О проекте

**PowerFit** — это готовый одностраничный сайт (лендинг) для фитнес-клуба, разработанный с акцентом на конверсию и пользовательский опыт.

### ✨ Особенности
- 🎨 Современный дизайн с оранжево-тёмной цветовой схемой
- 📱 Полностью адаптивная вёрстка (мобильные, планшеты, десктоп)
- ⚡ Плавная прокрутка к секциям
- 🎭 Анимации появления элементов при загрузке
- 📝 Валидация формы с маской телефона
- 🍔 Адаптивное мобильное меню
- 🖼️ Секция галереи с градиентными заглушками
- 💰 Тарифные карточки с выделением популярного плана

## 🚀 Быстрый старт

### Вариант 1: Локальный запуск
1. Скачайте файл `index.html`
2. Откройте его в любом браузере
3. Готово! 🎉

### Вариант 2: Через VS Code Live Server
1. Установите расширение "Live Server" в VS Code
2. Нажмите правой кнопкой на index.html → "Open with Live Server"

### Вариант 3: Деплой на GitHub Pages
1. Создайте репозиторий на GitHub
2. Загрузите файл index.html
3. В настройках репозитория включите GitHub Pages
